# Song Popularity Prediction
AI-Driven Song Popularity & Trend Prediction System. Here Song Popularity is predicted Using Audio Features and User Engagement Data

## Objective
Predict song popularity using Spotify audio features.

## Problem Statement
Music streaming platforms need to predict whether a song will become popular.
This project uses audio features and metadata to predict a song’s popularity score using Machine Learning.

## Dataset
Spotify Tracks Dataset (Kaggle)<br>
**Source: Kaggle**<br>
**Records: ~120,000 songs**

## ML Task
Supervised Regression

## Project Structure
- data/: datasets (raw and processed)
- notebooks/: entire analysis and model building (jupyter notebooks)
- src/: reusable code (python)
- models/: trained models
- reports/: documentation
